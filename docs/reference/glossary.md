# Glossary

The canonical repository glossary is [Concepts and Glossary]({{ site.baseurl }}/CONCEPTS/). The same core terms are mirrored in the in-app **Help (?) → Glossary** panel.

Key terms include Workload, Architecture, Assessment, Deep Investigation, Proactive Support, AMBA/Monitoring Coverage, Telemetry Coverage, Backup & DR Coverage, Retirement Radar, Evidence Locker, MCP, Connection, Sub Agent, Workbook, Playbook, and Demo data.

When writing cases or procedures, use these definitions consistently and include the scoped workload/connection and evidence timestamp rather than relying on an ambiguous resource nickname.
